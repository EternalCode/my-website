Website Demos
Experience some HTML5 Javascript games I've made

Dilshan Gunasekera




Snake


An implementation of the classic Snake arcade game. It's worth noting that there is not really a good way to control the snake on your phone, so I recommend the use of a keyboard HUD.








Pong


A more realistic Pong game. The Physics are a little closer to what one may expect in a realistic version of 2D pong. Utilizing vectors and paddle acceleration for ball pathing, this implementation may cause a little bit of troubles learning, but ball curving and control is more prominent. However, the X component of the ball's vector is a set amount to keep the gameplay and difficulty in check.






Stormy Rain


Just a cool visual effect of some rain drops drawn and animated in Javascript. I would take this example further by adding clouds, a splash animation and some thunder/lightning, but I'm not sure how worthwhile that'd be. Fair warning, this is a fullscreen animation, and not embedded on the page like the rest of the demos. Opening the watch link in a new tab would be ideal if you wish to save your progress in the other games.






Circle Game


There was this quite addicting circle "eating" game I found at http://circle-game.sysach.com. This is sort of my implementation of that game, with a few tweaks for difficulty levels. Again, fair warning, this is a fullscreen game and not embedded to the page.
